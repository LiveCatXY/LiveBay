# 【畅想】新社会畅想概要

## 一、以人为本还是以资为本

1. 人类基因的长期固定性——仁心永恒。
2. 人类社会的长期稳定性——为何要避免核扩散？
3. 资本主义经济学的范畴解构——私有财产神圣不可侵犯吗？
4. 资本主义对人类的异化和侵犯——要如何对待服务员？

## 二、如何管控资本

1. 资本只能作为工具或家畜进行管控。
2. 资本和政府的权力范围。
3. 是否要对资本增殖的限制。
4. 如何对资本增殖进行限制。

## 三、如何尽量把人从资本中解放出来

1. 尽量从事适合人类的创造性、创意性、家庭性工作。
2. 公平和效率的平衡。
3. 提高全民所有制下的保障和分红。
4. 建立真正以人为本的社会核心指标。

## 四、以人为本的经济学构想

1. 以人为本重构价值——生孩子算不上价值。
2. 基于人类本心的道德经济学——对穷人的扶持。
3. 如何确保以人为本的经济学的现实可行性——懒人如何激励？
4. 如何整合现有资本主义经济学。
