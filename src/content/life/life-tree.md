我给自己的大体上的人生架构取个名字，叫人生树：

```
//新新新\\
\\新新新//
  ｜业｜
  ｜智｜
//体心家\\
```

“体心家“是基本，“智业“是主干，“新“是轻果。保障基本，专业主干，逍遥轻果，一保二专三逍遥。

<details><summary>更多详情</summary>

1. 体：健康、体能
2. 心：心灵、道德
3. 家：家庭、政治
4. 智：智能、技术
5. 业：事业、商业、专业
6. 新：创新、创意、异态

也可以叫六边形战士🐶：

```
   新
智/‾‾\业
心\__/家
   体
```
</details>
