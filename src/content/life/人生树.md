我给自己的大体上的人生架构取个名字，叫人生树：

```
//新新新\\
\\心心心//
  ｜业｜
  ｜智｜
 //体家\\
```

1. “体家“是根基，🤍基础性内在结构。
2. “智业“是主干，💙支撑性中坚结构。
3. “心新“是茂叶，💛轻灵性外在结构。

保障根基，专业主干，逍遥茂叶，一保二专三逍遥。

也可以叫六边形战士🐶：

```
   新
智/‾‾\业
心\__/家
   体
```
